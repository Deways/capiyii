Capiyii

Deploying Yii Framework Applications with Capistrano

View project onGitHub

Welcome to GitHub Pages.

First, install Capiyii itself with RubyGems:

gem install capifony

To prepare your project for Capiyii, go into the root directory of that project and run capiyi:

$cd my_project
$capify .

Fill up your config/deploy.rb (or protectec/config/deploy.rb) with your server connection data. But first, you must choose your deployment strategy.

# deploy.rb

set   :application,   "My App"
set   :deploy_to,     "/var/www/my-app.com"
set   :domain,        "my-app.com"

set   :scm,           :git
set   :repository,    "ssh-gitrepo-domain.com:/path/to/repo.git"

role  :web,           domain
role  :app,           domain, :primary => true

set   :use_sudo,      false
set   :keep_releases, 3
set   :yii_console,  "www/protected/yiic"
